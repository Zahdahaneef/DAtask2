📊 Grade Calculator (Python)

This project is a simple Grade Calculator built using Python. It takes user input (marks) and displays the corresponding grade along with a message.

🚀 Features:

* Accepts user input for marks (0–100)
* Validates input using a loop
* Assigns grades based on score:
 
* 🟢 A (90–100)
* 🔵 B (80–89)
* 🟡 C (70–79)
* 🟠 D (60–69)
* 🔴 F (<60)

* Displays motivational messages for each grade
* Handles invalid input gracefully

🛠️ Technologies Used:
* Python 🐍
* Google Colab

🎯 Learning Outcomes: 

* Understanding Python functions
* Using conditional statements (if-elif-else)
* Implementing loops (while)
* Handling user input
* Writing clean and structured code
